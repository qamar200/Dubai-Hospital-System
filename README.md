# Dubai Hospital Insurance Authorization Portal 🇦🇪

A specialized healthcare software prototype designed to streamline insurance authorization processes in UAE-based hospitals.

## 🚀 Key Features
* **Automated Status Tracking**: Real-time visualization of insurance validity (Active/Expired).
* **Modern UI**: Professional dashboard interface built for hospital administrative efficiency.
* **Backend Logic**: Developed using .NET and C# to handle patient data and insurance providers.

## 🛠 Tech Stack
* **Language**: C#
* **Framework**: .NET Core
* **Frontend**: HTML5 & CSS3
